# devstudio

Creamos productos digitales que importan. Código limpio, diseño cuidadoso y soluciones escalables.

## Acerca de

devstudio es una landing page moderna diseñada para showcasear servicios de desarrollo web, aplicaciones móviles, consultoría y diseño UI/UX. Construida con tecnologías modernas y un enfoque en la experiencia del usuario.

## Características

- Diseño responsivo y moderno
- Paleta de colores personalizada
- Tipografía Rubik integrada
- Componentes reutilizables
- Estructura semántica y accesible

## Servicios

- Desarrollo Web
- Apps Móviles
- Consultoría
- UI/UX Design

## Tecnologías

- HTML5
- Tailwind CSS
- JavaScript
- Responsive Design

## Instalación

1. Clona este repositorio
2. Navega a la carpeta del proyecto
3. Abre `index.html` en tu navegador
